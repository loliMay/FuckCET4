## 无准考证原理分析
### 准考证规律
准考证号一共是15位
1. 前6位是省份、校园、校区代码 [点此查你的学校的前6位码]
2. 第7-8位是考试年份，例如2010年的即是10
3. 第9位是该年中第几次四六级考试，例如2010年12月份的考试由所以第2次考试，所以是2
4. 第10位是四六级种类，四级是1，六级是2
5. 第11-13位是考场号，例如第1考场即是001
6. 第14-15位是坐位号，例如1号坐位即是01
